## network related thing

a collection of networking tools for macos.

---

## how to use

<details>
<summary><b>running source</b></summary>

### requirements

download and extract the source code [here](https://github.com/countervolts/network-related-thing/archive/refs/heads/macos.zip)

-   python 3.8 or higher
-   macos (tested on 12.7.6 (Monterey))

### running source

```sh
python3 -m pip install -r requirements.txt

sudo python3 server.py
```
